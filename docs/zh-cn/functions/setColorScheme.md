`setColorScheme` 函数用于根据提供的十六进制颜色值，设置配色方案。

配色方案的设置会影响到页面中的所有 mdui 组件。详见 [动态配色](/zh-cn/docs/2/styles/dynamic-color)。

## 使用方法 {#usage}

按需导入函数：

```js
import { setColorScheme } from 'mdui/functions/setColorScheme.js';
```

使用示例：

```js
setColorScheme('#ff0000');
```
